# Logos

Place company logos here for use in PDF reports and the app.

## TIHLO logo

- **File name:** `tihlo-logo.png` (or `tihlo-logo.jpg`)
- **Usage:** Shift report PDF and other reports can use this as the default header logo when no tenant logo is set.
- **Format:** PNG or JPG; recommended size ~200–400px width for sharp printing.

Copy your TIHLO logo (gear + map pin) into this folder as `tihlo-logo.png` to have it appear on generated shift reports when the tenant has no custom logo.
